Windows® Installer Internet Download Bootstrapper

Setup.exe is a configurable bootstrap executable for Microsoft® Windows® Installer version 2.0. This provides an easy way to publish Windows® Installer packages on the web. This sample is a Win-32 based application which can be configured to download and install a package from the web.

The features supported by this sample include:

  1. Verification of the digital signature on the package on the web.
  2. Ability to upgrade Windows® Installer on the system to the latest version. It will also install the latest version if there isn't one on the system already.
  3. Ability to perform administrative installs.
  4. Ability to apply patches. It can also be used to patch an existing administrative install.

Building the sample

The sample can be built by running the following command from a Platform SDK build environment window: 

	nmake
In order to build the sample clean, the following commands should be used: 
	nmake clean
	nmake

Using the internet download bootstrapper

Msi.chm is the primary source of information about the Windows® Installer, and contains detailed reference material for developers. See the topics "Internet Download Bootstrapping" and "Msistuff.exe" for information on how to to use the internet download bootstrapper.

Limitations

This sample is not supported on 64-bit platforms. It is only designed to work on 32-bit Windows platforms. It also requires Internet Explorer version 4.00 or higher to be installed on the system in order to run.
